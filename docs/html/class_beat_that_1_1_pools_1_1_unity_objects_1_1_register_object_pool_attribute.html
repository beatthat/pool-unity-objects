Detailed Description

Convenience attrbibute for registering an UnityObjectPool<T> class.

Allows you to do this:

[RegisterObjectPool] // auto registers service as IObjectPool<MyObjectType> and IObjectFactory<MyObjectType> public class MyObjectFactory : UnityObjectPool<MyObjectType> {}

...instead of the longer, boiler-plate version [ResisterService( proxyInterfaces: new Type[] { typeof(IObjectPool<MyObjectType>), typeof(IObjectFactory<MyObjectType>) } )] public class MyObjectPool : UnityObjectPool<MyObjectType> {}
