◆ GetInstance()

T BeatThat.Pools.UnityObjects.PrototypeObjectFactory< T >.GetInstance ( )
inline

Creates a new instance of the prototype object. This implementation is non virtual because of IOS issues with virtual methods that have generic return type. If the new object requires configuration beyond what can be copied from the prototype, override InitInstance(object) for that purpose.

Returns
A new copy of the factory prototype.
